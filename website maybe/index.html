FreakyBrowse was built using Python and PyQt6 (originally PyQt5) by the Freakybob Team. It's what we want, and need, in a browser.

Features

- Notes (images & text!)

- Tabs (double-click to add a tab!)

- FreakySearch home page

- Themes (Added by us!)

- Bookmarks

- Save page

- View page source
